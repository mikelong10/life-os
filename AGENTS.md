# life os (amman)

## Tech Stack

- **Frontend:** React 19, Vite 7, TanStack Router (file-based routing)
- **Backend:** Convex (real-time database, server functions)
- **Auth:** BetterAuth with Convex integration, Google OAuth
- **Styling:** Tailwind CSS 4, Radix UI primitives, shadcn/ui components
- **Icons:** Lucide React
- **Charts:** Recharts
- **Hosting:** Vercel (SPA mode)

## Rules

- Always use `bun` as the package manager (not npm/yarn/pnpm)
